hooks.delete_files_and_folders

hooks.delete_files_and_folders(paths: Path | str | List[Path] | List[str]) → None

Delete files and folders for given file and/or folder paths.

Parameters:

  • paths – A pathlib.Path object or string, or list of pathlib.Path or

  • strings.

Returns:

None - deletes the files and/or folders defined in paths if they exist. If this function raises an error, the files and/or folders have not been removed.
